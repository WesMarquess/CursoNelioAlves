📚 Repositório de Estudos – Curso do Professor Nélio Alves

Este repositório contém os códigos, exercícios e projetos desenvolvidos durante as aulas do curso ministrado pelo professor Nélio Alves.

O objetivo deste repositório é organizar e documentar a evolução dos estudos ao longo do curso, servindo como material de apoio e consulta futura.

🚀 Conteúdo Abordado

Nos exercícios e projetos presentes neste repositório, são trabalhados os seguintes conceitos e tecnologias:

🧱 Programação Orientada a Objetos (POO)

Classes e Objetos

Encapsulamento

Herança

Polimorfismo

Interfaces

🌱 Spring Framework

Injeção de Dependência

Criação de APIs REST

Spring Boot

Camadas (Controller, Service, Repository)

🗄️ Banco de Dados

Modelagem Relacional

SQL

Integração com aplicações Java

JPA / Hibernate

🎯 Objetivo

Consolidar conhecimentos em desenvolvimento backend com Java, aplicando boas práticas de programação, arquitetura em camadas e integração com banco de dados.

📌 Observação

Este repositório tem fins exclusivamente educacionais, sendo utilizado como apoio prático às aulas do curso.
